¡Caballeros, contemplen! Esto es Strongly Typed!

Está bien, ¿qué es esto?

Esto es Strongly Typed, un sitio web HTML5 gratuito y completamente receptivo creado por HTML5 UP. Gratis para uso personal y comercial bajo la licencia CCA 3.0.

¡Genial! ¿Qué es HTML5 UP?

HTML5 UP es un proyecto paralelo de AJ (es decir, yo). Comencé como una forma de probar mis herramientas receptivas y mejorar mis habilidades de codificación y diseño.

¿Con qué está construido esto?

Responsive Tools es un conjunto simple de herramientas para construir sitios web y aplicaciones receptivos. Todos mis plantillas en HTML5 UP están construidas utilizando estas herramientas.
